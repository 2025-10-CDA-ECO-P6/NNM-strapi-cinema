##  TMDb API

### Endpoint : Récupérer les films populaires
**Méthode :** `GET`  
**URL :** `http://localhost:1337/api/tmdb/popular`  
**Authentification :** Non requise (`auth: false`)  
**Description :** Retourne la liste des films populaires depuis l’API TMDb.

### Exemple avec `curl` :
```bash
curl -X GET http://localhost:1337/api/tmdb/popular
```
